> **논문 정보**
>
> - **제목**: A-MEM: Agentic Memory for LLM Agents
> - **저자**: Wujiang Xu, Zujie Liang, Kai Mei, Hang Gao, Juntao Tan, Yongfeng Zhang (Rutgers University, AIOS Foundation)
> - **출판**: NeurIPS 2025 / arXiv 2502.12110 (2025.02)

시리즈의 마지막 논문이다. 스물다섯 편의 여정을 마무리하기에 이보다 적절한 주제가 없다 — 기억.

시리즈의 첫 글에서 CoALA가 에이전트의 세 축 중 하나로 기억을 꼽았다. Reflexion에서 반성이 에피소드 메모리에 저장되었고, LATS에서 트리 구조가 외부 메모리로 기능했다. 하지만 이 모든 경우에 메모리의 구조는 사전에 정의되어 있었다 — 슬라이딩 윈도우, 키-값 쌍, 트리 노드.

A-MEM은 다른 질문을 던진다. 에이전트가 스스로 메모리의 구조를 만들 수 있는가?

영감의 원천은 니클라스 루만의 제텔카스텐(Zettelkasten)이다. 독일의 사회학자가 70권의 책과 400편의 논문을 쓸 수 있었던 비결로 알려진 메모 시스템. 각 메모는 독립적인 단위이지만, 메모들 사이에 링크가 형성되어 지식의 네트워크가 자연스럽게 성장한다.

A-MEM은 이 원리를 세 가지 메커니즘으로 구현한다. 노트 구성(Note Construction)은 상호작용을 키워드, 태그, 맥락적 설명, 임베딩이 포함된 구조화된 노트로 변환한다. 링크 생성(Link Generation)은 유사도 검색과 LLM 분석으로 메모리 간 의미적 연결을 자율적으로 설정한다. 메모리 진화(Memory Evolution)는 새 메모리가 추가될 때 기존 메모리의 맥락, 키워드, 태그를 동적으로 업데이트한다.

결과가 인상적이다. 6개 기반 모델에서 기존 SOTA(LoCoMo, MemGPT, ReadAgent, MemoryBank)를 일관되게 능가하며, Multi-Hop 추론에서 2배 이상 성능이 향상된다. 토큰 사용량은 85-93% 절감(1,200~2,500 vs 16,900 토큰)이다. 더 적은 토큰으로 더 정확한 기억을 해낸다.

### CoALA 좌표계의 완성

시리즈의 첫 글에서 세운 좌표계로 돌아간다. A-MEM은 기억 축의 진화를 보여준다.

| 시리즈 논문 | 기억 방식 |
|-----------|---------|
| CoALA | 기억의 존재를 정의 |
| Reflexion | 반성 텍스트를 슬라이딩 윈도우에 저장 |
| LATS | 트리 구조로 탐색 이력 보존 |
| ETO | 궤적 쌍을 모델 가중치에 내재화 |
| A-MEM | 자율적으로 구조를 만들고 진화시키는 기억 |

에이전트의 기억은 고정된 구조에서 동적 구조로, 외부 저장에서 자기 조직화로 진화해왔다.

### 시리즈를 마치며

스물다섯 편의 논문을 통해 에이전트 AI의 지형을 걸었다. 추론의 씨앗(CoT)에서 행동과의 결합(ReAct)으로, 도구 사용(Toolformer)에서 다중 에이전트(AutoGen, MetaGPT)로, 자기 반성(Reflexion)에서 체계적 탐색(LATS)으로, 모델 훈련(ETO)에서 비판적 평가(AI Agents That Matter)로, 금융 특화(BloombergGPT~FINCH)에서 안전성(Constitutional AI, RLHF)으로, 그리고 마침내 기억(A-MEM)으로.

이 여정에서 반복적으로 나타난 패턴이 있다. 에이전트는 혼자보다 함께일 때 강하고, 성공보다 실패에서 더 많이 배우고, 복잡한 구조보다 정확한 평가가 더 중요하고, 범용성과 전문성 사이의 균형이 늘 설계의 핵심이다.

논문은 여기서 끝나지만, 에이전트의 진화는 계속된다.
